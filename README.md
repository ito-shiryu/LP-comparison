\# LP差分確認ツール



既存LPと変更後LPの画像を比較し、変更点を視覚的に明示するツールです。



\## 機能

\- スライダー比較（ドラッグで左右切り替え）

\- 差分ハイライト（ピクセル差分を赤でオーバーレイ）

\- 並列比較（左右並べて確認）

\- AI解析（Claude Vision APIで変更内容を日本語で自動生成）

\- PNG出力（番号付き枠＋変更点テキスト付き）



\## 開発環境のセットアップ



\\`\\`\\`bash

npm install

npm run dev

\\`\\`\\`



\## デプロイ（Vercel）



Vercelにリポジトリを連携するだけで自動デプロイされます。

ビルドコマンド: `vite build` / 出力ディレクトリ: `dist`
